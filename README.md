### Overview
The Interactive Personal Data Collector is a command-line interface (CLI) application that prompts users for personal details and provides a technical breakdown of how that data is stored in the system's memory.

### Features
User Interaction: Collects name, age, height, and a favorite number through interactive prompts.

Data Transformation: Converts standard string inputs into appropriate numerical types (Integer and Float) for calculation.

Logic & Calculation: Automatically estimates the user's birth year based on the current year (2026) and the provided age.

Technical Insights: Displays the Python data type and the specific memory address (id) for every variable collected.

### Technical Details
The script demonstrates the use of several core Python functions:

input(): For capturing user data as strings.

int() & float(): For explicit type casting.

type(): To verify the class of the object.

id(): To reveal the identity/memory location of the object.
